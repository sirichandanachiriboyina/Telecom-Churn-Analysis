
# 📊 Telecom Customer Churn Analysis

## Overview
This project analyzes customer churn behavior in the telecom industry using **Python, SQL, Machine Learning, and Tableau** to identify factors contributing to customer attrition and recommend strategies to improve customer retention.

The project combines **Exploratory Data Analysis (EDA)**, **Machine Learning modeling**, **SQL analysis**, and an **interactive Tableau dashboard** to generate actionable business insights.

---

## 🎯 Business Objective
Telecom companies face customer loss due to competition and changing customer preferences.

This project aims to:

- Identify key factors driving customer churn
- Predict customer churn using machine learning
- Analyze customer behavior patterns
- Provide business recommendations to improve retention

---

✔ Correlation Analysis  
✔ Churn Pattern Analysis  
✔ Customer Segmentation

### Key EDA Findings
- Customers with **month-to-month contracts churn the most**
- Customers using **electronic check payment methods show higher churn**
- **Higher monthly charges** increase churn probability
- Customers with **shorter tenure** are more likely to churn
- **Online security and tech support services reduce churn**

## 💡 Business Recommendations
1. Encourage customers to switch from **month-to-month contracts** to long-term contracts.
2. Improve customer experience for **electronic check users**.
3. Create retention programs for **new customers**.
4. Bundle **tech support and online security services**.
5. Offer pricing incentives for customers with **high monthly charges**.
